商品后台管理

这个后台用 Supabase 表格控制，最稳。你改完数据库，前台会自动读取。

怎么下架商品

  1. 打开 Supabase → Table Editor → products
  2. 找到要下架的商品
  3. 把 active 从 TRUE 改成 FALSE
  4. 刷新前台网站，商品就不会显示

打开 Supabase 商品表

能改什么

  • name：商品名
  • price：价格
  • sales：已售数量
  • active：TRUE 上架 / FALSE 下架
